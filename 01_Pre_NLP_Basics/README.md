# Pre_NLP_Basics
## 传统基础知识
- 自动分词：将连续文本切分为词或词组，常用于中文等无空格语言。方法包括基于规则、基于统计（如HMM/CRF）和基于深度学习（如BiLSTM、Transformer）。常见工具：jieba、THULAC、HanLP。
- 命名实体识别（NER）：识别文本中的特定实体并分类（如人名、地名、机构名等），常用序列标注方法（CRF、BiLSTM-CRF）或预训练模型（BERT）微调。
- 词性标注：为词分配词性标签（名词、动词等），是句法分析和信息抽取的基础。常用方法包括HMM、CRF和神经网络模型。
- 句法分析：分析句子内部词之间的结构关系，常见形式为成分句法树或依存句法树，用于理解句子结构与关系。
- 语义分析：理解句子或词的含义，包含词义消歧、语义角色标注、语义依存等；现代方法多借助预训练语义表示并结合下游任务微调。
- 篇章分析：是对整篇或多段文本进行的综合分析，核心是理解文本的整体结构、语义逻辑和信息关联。
## PyTorch实现回归实验
- PyTroch张量(Tensor) 和 NumPy数组(Array)

Tensor:深度学习优化,支持GPU计算,自动求导(内置Autigrad),数据类型(高精度)。
### 1. 线性回归基础
线性回归的核心是拟合线性关系 $y = wx + b$，其中：
- $w$：权重（斜率）
- $b$：偏置（截距）
- 目标：找到最优的 $w$ 和 $b$，使预测值与真实值的误差最小。

### 2. 神经网络构成
本次实验的神经网络仅包含一层线性层（`nn.Linear`），对应：
- 输入层：1 维（x 的维度）
- 输出层：1 维（y 的维度）
- 无隐藏层，是最简单的神经网络结构。

### 3. 训练流程
1. **前向传播**：输入数据通过模型得到预测值；
2. **损失计算**：用均方误差（MSE）衡量预测值与真实值的差距：
   $$MSE = \frac{1}{n}\sum_{i=1}^n (y_{pred} - y_{true})^2$$
3. **反向传播**：计算损失对参数 $w$、$b$ 的梯度；
4. **参数更新**：用随机梯度下降（SGD）更新参数，最小化损失。
